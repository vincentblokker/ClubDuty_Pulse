# Task ID: 4
# Title: Implement Round Management
# Status: pending
# Dependencies: 2, 3
# Priority: high
# Description: Develop functionality for coaches to create, open, close, and manage feedback rounds.
# Details:
1. Create Round controller with CRUD operations
2. Implement endpoints:
   - POST /api/rounds (create new round)
   - GET /api/rounds (list rounds for a team)
   - GET /api/rounds/:id (get round details)
   - PUT /api/rounds/:id (update round)
   - PUT /api/rounds/:id/status (open/close round)
3. Build round creation form in frontend
4. Implement round list view for coaches
5. Create round detail view with status controls
6. Add validation for round dates and status changes
7. Implement round status lifecycle (draft, open, closed)
8. Create unique URL generation for sharing with players

# Test Strategy:
Test round creation, opening, and closing. Verify status transitions and validation rules. Test URL generation and access control. Ensure rounds are properly associated with teams.
